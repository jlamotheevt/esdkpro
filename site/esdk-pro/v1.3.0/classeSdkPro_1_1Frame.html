◆ GetFrameId()

FrameId_t eSdkPro::Frame::GetFrameId ( ) const

The unique incrementing id of this frame. May be used as a frame counter. The first frame received will have an id of 0. In the case where there are missing frames due such as when transmission errors occur, there will be gaps between subsequent frame ids.

◆ GetTimestampNs()

TimestampNs_t eSdkPro::Frame::GetTimestampNs ( ) const

The timestamp in nanoseconds (ns) of this frame. For frames received from the camera, the GVSP timestamp is converted to nanoseconds using the camera's timestamp tick frequency. When using PTP, the timestamp will be based off of the epoch of the master clock.
